Framework est plate-formes supportées

JCAssertion est compilé pour le .net framework 4.5. Comme il contient directement une dll de oracle pour l'accès aux bases de données oracle il n'est par re commandé de l'intégrer directement à des projets avec un autre framework. Une versuion pour des versions supérieures de framework pourrait être faite si quelqu'un manifeste de l'intérêt.
