Configuration requise

JCAssertion a été testé sous windows 7 et windows 8.1.

Pour que JCAssertion fonctionne le framework .NET 4.5 de microsoft doit être installé.

Pour que le programme puisse écrire des messages d'erreurs dans le journal des événements de windows, on doit créer la source 'JCAssertion' dans le journal des événements. Pour faire cela il faut lancer en tant qu'administrateur le fichier de commande windows 'creersource.cmd' qui utilisera un programme standard de windows pourfaire cette création.  Ce fichier de commande est avec les autres programmes de JCAssertion.
