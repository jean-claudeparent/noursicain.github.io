Installation de JCAssertion

À partir de la version 2.0.0 JCAssertion est disponible sous forme de nuget. On est cependant capable de l'obtenir sans utiliser un environnement de développement ou un gestionnaire de nuget. La marche à suivre pour installer sans IDE est:

  1. Aller sur nuget gallery à l'adresse: https://www.nuget.org/packages/org.jcassertion.JCAssertion/
  2. Dans la section "Info" cliquez sur "Download package" et sauvegarder le fichier sur votre ordinateur.
  3. Trouvez le fichier sur votre ordinateur, lr nom ressemblera à org.jcassertion.JCAssertion.2.0.0.nupkg. Comme le numéro de version est dans le nom de fichier il pourra être différent du nom écrit plus haut. Renommez le fichier en fichier zip par exemple org.jcassertion.JCAssertion.2.0.0.zip
  4. Ouvrez le fichier zip et copiez le contenu du répertoire "content" et du répertoire "lib\met45" dans un répertoire sur votre disque. JCAssertion est maintenant disponible pour être utilisé avec les progrannes en ligne de commande JCAssertion.exe et JCAExport.exe.

Naturellement jcassertion peut être utilisé comme un nuget normal dans un projet visual studio.

Le code source de JCAssertion peut être obtenu en téléchargeant JCAssertion à partir de la page de "release" sur Github (nécessite un compte Github gratuit).

Assurez-vous d'avoir la configuration requise. Il est toujours bon d'avoir un bon anti virus actif sur votre ordinateur.
