Installation de JCAssertion

La version courante de JCAssertion est la version 1.0.4. Elle peut être obtenue en cliquand sur télécharger JCAssertion

Une fois le fichier téléchargé, extraire tous les fichiers du dossier compressé et les mettre dans un répertoire comme par exemple: c:outils JCAssertion.

Assurez-vous d'avoir la configuration requise. Il est toujours bon d'avoir un bon anti virus actif sur votre ordinateur.
