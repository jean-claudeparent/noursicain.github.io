DocDocos exemple d'utilisation

Utilisation simple - C#

Dans son utilisationla plus simple on indique à la composante quel fichier xml lire et dans quel répertoire on met les pages du site.

using DocDocos;
...
DocDocosDA Generateur =
         new DocDocosDA();

Generateur.FichierXMLDoc = @"c:\monfichier.xml";
Generateur.RepertoireSortie = @"c:\mon site";
Generateur.GenererHTML();

Utilisation avec gabarit optionnel - C#

On peut spécifier un gabarit dans le constructeur. À ce moment les pages générées aurons la même extension

using DocDocos;
...
DocDocosDA Generateur =
         new DocDocosDA(@"c:\mongabarit.php");

Generateur.FichierXMLDoc = @"c:\monfichier.xml";
Generateur.RepertoireSortie = @"c:\mon site";
Generateur.GenererHTML();
