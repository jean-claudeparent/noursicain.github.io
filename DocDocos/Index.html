org.jcassertion.DocDocos

DocDocos est une composante nuget permettant de créer un site web simple. Le site est créé à partir du fichier xml de documentation généré par Visual studio.

Dans son utilisationla plus simple on indique à la composante quel fichier xml lire et dans quel répertoire on met les pages du site.

Il est possible de modifier un peu la page html qui sert de modèle à la création des autres pages. Pour cela il faut copier le fichier DocDocos-gabarit.html dans le même répertoire que la dll de DocDocos et de modifier ce fichier. Par exemple pour mettre une référence à une feiille de style.
