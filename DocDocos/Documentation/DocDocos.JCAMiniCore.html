DocDocos.JCAMiniCore

Sommaire

Le mini core contient des fonctions qui ont juste des dépendances avec le framework .net

DocDocos.JCAMiniCore.EffaceFichier(System.String) Efface un fichier s'il existe
DocDocos.JCAMiniCore.NomValide(System.String) Remplace les catactères invalides dans un nom de fichier par des soulignées Exemple "Fichier:01" deviendra "Fichier_-1"
DocDocos.JCAMiniCore.TypeEncodage(System.String) Retourne l'encodage d'un fichier texte en se basant sur le BOM (octets d'identification au début du fichier) Si le BOM est manquant ou inconnu ASCII est assumé
