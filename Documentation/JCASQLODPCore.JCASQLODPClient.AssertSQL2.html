JCASQLODPCore.JCASQLODPClient.AssertSQL(System.String,System.String)

Sommaire

AssertSQLString (CommandeSQL,ResultatAttendu): Fait un select retournant un texte sur la connection courante et retourne si la valeur est égale au Resultat attendu. Si aucune rangée n'est retournée par le select on retourne false
