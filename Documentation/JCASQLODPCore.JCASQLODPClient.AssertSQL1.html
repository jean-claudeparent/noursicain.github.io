JCASQLODPCore.JCASQLODPClient.AssertSQL(System.String,System.Double,System.String)

Sommaire

AssertSQL (CommandeSQL,ResultatAttendu): Fait un select retournant un nombre sur la connection courante et retourne si la valeur est égale (ou est évaluée avec un autre operateur) au Resultat attendu. Si aucune rangée n'est retournée par le select on retourne false. Modifie aussi la propriété DernierResultat
