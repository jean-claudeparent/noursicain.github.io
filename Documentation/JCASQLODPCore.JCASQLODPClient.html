JCASQLODPCore.JCASQLODPClient

Sommaire

JCASQLODPCore : Classe définissant les propriétés et méthodes pour accéder une base de données oracle avec le oracle data provider.

JCASQLODPCore.JCASQLODPClient.AssertSQL(System.String,System.Double,System.String) AssertSQL (CommandeSQL,ResultatAttendu): Fait un select retournant un nombre sur la connection courante et retourne si la valeur est égale (ou est évaluée avec un autre operateur) au Resultat attendu. Si aucune rangée n'est retournée par le select on retourne false. Modifie aussi la propriété DernierResultat
JCASQLODPCore.JCASQLODPClient.AssertSQL(System.String,System.String) AssertSQLString (CommandeSQL,ResultatAttendu): Fait un select retournant un texte sur la connection courante et retourne si la valeur est égale au Resultat attendu. Si aucune rangée n'est retournée par le select on retourne false
JCASQLODPCore.JCASQLODPClient.ChargeLOB(System.String,System.String) ChargeLOB : Charge un Large object dans toutes les rangées définies par un énoncé SQL.
JCASQLODPCore.JCASQLODPClient.CreerConnectionString CreerConnectionString : retourne la chaîne de connection crée à partir des propriétés de la classe. Peut lever une exception.
JCASQLODPCore.JCASQLODPClient.ExporteLOB(System.String,System.String,System.Text.Encoding) ExporteLOB : Écrit des colonnes BLOB ou CLOB dans des fichiers.
JCASQLODPCore.JCASQLODPClient.FermerConnection FermerConnection: Ferme la connection à ka vase de dibb.es en utilisant ODP. Peut retourner des exceptions.
JCASQLODPCore.JCASQLODPClient.OuvrirConnection OuvrirConnection: Ouvre la connection à ka vase de dibb.es en utilisant ODP. Peut retourner des exceptions.
JCASQLODPCore.JCASQLODPClient.Resumer Resumer : Resume le datareader dans la propriété Resume Sert à débugger
JCASQLODPCore.JCASQLODPClient.SiConnectionOuverte SiConnectionOuverte retourne si la connection est ouverte
JCASQLODPCore.JCASQLODPClient.SQLExecute(System.String) SQLExecute: Execute une command SQL qui modifie la base de données sur la connection courante La commande est passé en paramètre. Retourne le nombre de rangées modifiées.
JCASQLODPCore.JCASQLODPClient.SQLSelect(System.String) SQLSelect: Fait un select sur la connection courante et met le resultat dans monReader. La commande est passé en paramètre.
