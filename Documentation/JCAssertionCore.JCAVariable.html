JCAssertionCore.JCAVariable

Sommaire

JCAssertionCore.JCAVariable.CleExiste(System.String) retourne si une clé existe dans le dictionnaire de variables
JCAssertionCore.JCAVariable.EcrireFichier(System.String) Écrit le dictionnaire des variables comme un document xml dans un fichier xml. Le dictionnaire est trié.
JCAssertionCore.JCAVariable.EnleverCle(System.String) Enlève une clé et sa valeur du dictionnaire de variables. Si la clé n'existe pas il n'y a pas d'exception de lancée.
JCAssertionCore.JCAVariable.EstEgal(System.Collections.Generic.Dictionary{System.String,System.String},System.String@) Compare deux dictionnaires de variables.
JCAssertionCore.JCAVariable.ExtrairePaire(System.String,System.String@) retourne la clé et dans une variable de sortie la valeur d'une paire sous la forme cle=valeur si jamais la clause = valeur n'existe pas la valeur retournée sera la clé
JCAssertionCore.JCAVariable.ExtraireVariable(System.String) Retourne le nom de la prochaine variable trouvée dans le texte. Les variables sont délimitées par "{{" et "}}"
JCAssertionCore.JCAVariable.GetDictionnaireVariable retourne le dictionnaire des variables
JCAssertionCore.JCAVariable.GetValeurVariable(System.String) GetValeurVariable : retourne la valeur d'une clé identifiant une variable
JCAssertionCore.JCAVariable.LireFichier(System.String,System.Boolean) Lit un fichier xml définissant le dictionnaire de variable et le met dans le dictionnaire.
JCAssertionCore.JCAVariable.MAJVariable(System.String,System.String) Si la clé existe dans le dictionnaire de contenu de variables, la valeur est mise à jour. Sinon la clé et la veleur sont crées dans le dictionnaire. Le dictionnaire est ensuite trié.
JCAssertionCore.JCAVariable.NombreVariables Retourne le nombre de variables dans le dictionnaire.
JCAssertionCore.JCAVariable.SubstituerVariables(System.String,System.Collections.Generic.Dictionary{System.String,System.String}) Remplace toutes les balises de variable par le contenu du dictionnaire des valeurs de variables. Si la variable n'est pas dans le dictionnaire cela lance une exception.
JCAssertionCore.JCAVariable.Trier trie le dictionnaire des variables par ordre croissant alphabétique de clé.
