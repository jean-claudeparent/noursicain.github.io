JCAssertionCore.JCAPontXML.AssertCompteFichiers(System.String,System.String,System.String,System.Int64,System.Int64@)

Sommaire

Évalue une assertion sur le nombre de fichiers comptés dans un répertoire. Cette méthode est utilisée en passant des paramètres du bon type et non pas un document xml définisssant l'assertion.

Retourne

Si l'assertion est vraie ou fausse
