JCASQLODPCore.JCASQLODPHelper.MAJLOB(System.Data.DataSet@,System.String)

Sommaire

Met le contenu d'un fichier dans une des colonnes d'un dataset. Le dataset doit avoir une colonne nommée BLOB ou une colonne nommée CLOAB. Ceci est accompli en mettant un alias de colonne dans la commande SQL qui crée le dataset.

Retourne

Le nombre de rangées du dataset modifiées.
