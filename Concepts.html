JCAssertion : Concepts de base

Le concept est de définir dans un fichier xml un ensemble d'assertion qui si elles sont toutes vraies font que l'essai est réussit. Une assertion peut être une vérification comme par exemple de vérifier qu'un fichier contient une certaine valeur, ou une action, comme par exemple exécuter un programme. Il y a ddiverses balises définissant un type d'assertion et les paramètres de celle-ci. Aussi pour que les assertions restent générales et ré utilisables on peut utiliser un fichier de variables. Par exemple le fichier de variable peut définir quel est le répertoire où on trouvera les fichiers de données. Plus de détails sur les variables.

Le fichier de variable peut etre créé initialement avec le programme JCAExporte.exe. Quand le fichier est créé il peut être modifié avec un éditeur de fichier ou par des assertions lorsque celles-ci sont exécutées avec le programme JCAssertion.exe.
