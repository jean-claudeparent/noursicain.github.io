Tutoriel de JCAssertion

Tutoriel étape par Étape

Étape 1 : Créer un répertoire de travail

Créez un répertoire pour le tutoriel sur votre disque dur. Pour le tutoriel sur ce site ce répertoire sera D:\DevCenter\Tutoriel\JCAssertion. Si vous utilisez un répertoire différent nMiybliez pas de le menttre partout où le tutoriel utilise "D:\DevCenter\Tutoriel\JCAssertion".

Étape 2 : Installer JCAssertion

  • la configuration requise. Si ce n'est pas le cas installez les composantes manquantes.
