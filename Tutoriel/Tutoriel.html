Étape 1 : Créer un répertoire de travail

Créez un répertoire pour le tutoriel sur votre disque dur. Pour le tutoriel ce réperoire sera D:\DevCenter\Tutoriel\JCAssertuion. Si vous utilisez un répertoire différent nMiybliez pas de le enttre partout où le tutoriel utilise "D:\DevCenter\Tutoriel\JCAssertuion".
