Configuration de Oracle pour JCAssertion

Pour exécuter les essais de JCAssertion pour la partie accès aux base de données oracle if laut:

  1. En se connectant comme administrateur de la base de données, créer l'utilisateur oracle "JCA" avec le not de passe "JCA". La commande est: create user JCA identified by JCA;
  2. Donner à l'utilisateur la permission de créer des tables dans son propre shéma. La commande est : grant create table to JCA;
  3. Donner à l'utilisateur la permission de se connecter, la commande est : grant create session to JCA;
  4. Si l'instance de oracle n'a janaus eté configurée au niveau du "tablespace" il faut configurer le "tablespace". Pour savoir si c'est fait faire l'étape de création de table qui suit celle-ci et si on a une erreur de "tablespace" le table space n'est pas configuré. Voici un exemple de message d'erreur : ORA-01950: no privileges on tablespace 'SYSTEM'
  5. Se reconnecter à la base de données avec l'utilisateur JCA et le mot de passe JCA et créer les tables nécissaires aux essais.
