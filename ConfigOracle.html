Configuration de Oracle pour JCAssertion

Pour exécuter les essais de JCAssertion pour la partie accès aux base de données oracle if laut:

  1. En se connectant comme administrateur de la base de données, créer l'utilisateur oracle "JCA" avec le not de passe "JCA". La commande est:
    create user JCA identified by JCA;
  2. Donner à l'utilisateur la permission de créer des tables dans son propre shéma. La commande est :
    grant create table to JCA;
  3. Donner à l'utilisateur la permission de se connecter, la commande est :
    grant create session to JCA;
  4. L'instance par défaut de oracle express 11 g comporte un tablespace USERS déjà défini pour les tables utilisateur. Si vousutilisez une instance oracle avec le tablespace USERS défini, l'assigner comme tablespace par défaut pour l'utilisateur JCA avec la commande :
    alter user JCA default tablespace USERS;
  5.  Se reconnecter à la base de données avec l'utilisateur JCA et le mot de passe JCA et créer les tables nécissaires aux essais.

Commandes pour créer les tables

CREATE TABLE JCAUSER(
LOGON VARCHAR2(20) NOT NULL PRIMARY KEY,
NOM VARCHAR2(60) NOT NULL
);
