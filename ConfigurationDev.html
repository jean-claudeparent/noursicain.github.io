Configuration requise pour le développeur

Pour pouvoir modifier et compiler JCAssertion il faut: .

  • Microsoft Visual Studio express 2012 pour les applicatios windows
  • Microsoft Visual Studio express 2012 pour le web (pour modifier les pages web du site de documentation)
  • Github desktop pour windows pour travailler à partir du code source sur github

Pour exécuter les tests il faut:

  • Oracle express 11g x64
  • Configurer Oracle pour les essais JCAssertion
