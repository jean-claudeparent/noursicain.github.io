Fichier XML des assertions

Le fichier XML des assertions à la forme suivante:

<?xml version="1.0" encoding="utf-8" ?>
<ListeDAssertion>
 <Assertion>
  <Description>Vérifier si le fichier de résultat contient certaines informations. </Description>
  <Type>ContenuFichier</Type>
  <Fichier>{{Log}}</Fichier>
  <Contient>Le fichier existe.</Contient>
  <Contient>Nombre de cas à traiter : 3</Contient>
 </Assertion>
</ListeDAssertion>
