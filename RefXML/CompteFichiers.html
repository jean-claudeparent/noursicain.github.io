CompteFichiers

Description

CompteFichiers compte le nombre de fichiers qui se trouvent dans un répertoire.  Seuls les fichiers sont comptés, pas les répertoires. Si le répertoire contient des sous répertoires les fichiers contenus dans les sous répertoires sont comptés. Un pattern de fichiers peut être spécifié comme par exemple "a*.txt". Seuls les fichiers répondant au pattern serons comptés. Si l'utilisateur qui lance l'essai n'a pas les accès pour lire le contenu des répertoires l'assertion sera fausse. Tout autre erreur technique sera aussi considéré comme donnant une assertion fausse.

Balises

Nom Obligatoire Description
MessageEchec Non obligatoire  Message à véhiculer si l'assertion est fausse. .
Operateur Non obligatoire  Opérateur de comparaison à appliquer sur le résultat réel et le résultat attendu pour évaluer si l'assertion est vraie. Voir les les opérateurs de comparaison numériques . Si la balise n'est pas  spécifiée la valeur par défaut est 'PG' (Valeur réelle plus grande que valeur attendue).
Pattern Non obligatoire Pattern des fichiers à compter. Par exemple "a*.txt" comptera tous les fichiers dont le nom commence par "a" et dont l'extention est ".txt". Windos ne fait pas de distinction entre les majuscues et les minuscules. Si le pattern n'est pas spécifié sa valeur par défaut est "*.*". Si un pattern contient des cacractères invalides pour un nom de fichier comme par exemple ":" l'assertion sera considérée comme fausse.
Repertoire Obligatoire  Répertoire dans lequel on compte les fichiers.
ResultatAttendu Non obligatoire  Valeur à comparer avec le résultat réel pour déterminer si l'assertion est vraie. Doit contenir un nombre entier . Si la balise n'est pas spécifiée, la valeur par défaut est 0.

Exemple

 <Assertion>
  <Description>Y-a-il plus de 10 fichiers? </Description>
  <Type>CompteFichiers</Type>
  <Repertoire>{{Log}}</Repertoire>
  <Pattern>Si*.x*</Pattern>
  <Operateur>pg</Operateur>
  <MessageEchec>Il n'y a pas 10 fichiers</MessageEchec>
  <ResultatAttendu>10</ResultatAttendu>
 </Assertion>
