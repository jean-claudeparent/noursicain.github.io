FichierExiste

Description

FichierExiste vérifie si un fichier existe. Pour les version 1.0.4 et moins de JCAssertion un répertoire mis comme nom de fichier retourne toujours un échec. Dans la version 1.0.5 JCAssertion vérifie braiment que le répertoire existe.

Balises

Nom Obligatoire Description
Fichier Oui Nom du fichier dont il faut tester l'existence. Le chemin complet devrait être spécifié pour obtenir des résultats constants.
MessageEchec Non Message personalisé qui sera utilisé pour signaler que l'assertion et fausse.

Exemple

 <Assertion>
  <Description>Vérifier que le fichier de données existe. </Description>
  <Type>FichierExiste</Type>
  <Fichier>{{VariableContenantLeChemin}}FichierDeDonnees.xml</Fichier>
  <MessageEchec>Le fichier de données n'existe pas.</MessageEchec>
 </Assertion>
