MAJVariables

Description

MAJVariables permet de changer la valeur d'une variables identifiée par une clé. Ce changement se fait dans le fichier de variable si JCAssertion.exe utilise un fichier de variables, sinon le changement est fait uniqueent dans la table des variables en mémoire.

Balises

Nom Obligatoire Description
Cle Oui Clé identifiant la variable dont la valeur doit être changée. Si la variable n'existe pas elle sera créée.
Valeur Oui La nouvelle valeur de la variable.

Exemple

 <Assertion>
  <Description>Changer la valeur de la variable "bin". </Description>
  <Type>MAJVariables</Type>
  <Cle>bin</Cle>
  <Valeur>c:\app</Valeur>
  </Assertion>
