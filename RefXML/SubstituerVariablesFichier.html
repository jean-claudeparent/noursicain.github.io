SubstituerVariablesFichier

Description

SubstituerVariablesFichier lit un fichier texte modèle qui contient des balises de variables. Le fichier modèle est lu, les balises de variabkes sont remplacées par la veleur des variables et le résultat est écrit dans le fichier de sortie. Le fichier modèle n'est pas modifié. S'il manque une valeur de variable ou si une erreur est rencontrée l'assertion est considérée comme fausse.

Balises

Nom Obligatoire Description
FichierModele Oui Le nom du fichier de modèle, avec son chemin d'accès.
FichierSortie Oui Le nom du fichier de sortie, avec son chemin d'accès.
FichierVariables Oui Le nom du fichier de bariables, avec son chemin d'accès.

Exemple

 <Assertion>
  <Description>Créer un fichier sql pour un logon en particuler. </Description>
  <Type>SubstituerVariablesFichier</Type>
  <FichierModele>{{Chemin}}\ModeleSQL.sql</FichierModele>
  <FichierSortie>{{Chemin}}\{{User}}monSQL.sql\</FichierSortie>
  <FichierVariables>{{Chemin}}\Var.xml</FichierVariables>
</Assertion>
