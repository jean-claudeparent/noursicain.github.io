ContenuFichier

Description

ContenuFichier permet de vérifier si un fichier texte contient un ou plusieurs texte et si il ne contient pas un ouplusieurs textes. Typiquement on vérifie que les textes des résultats atendus sont dans le fichier et qu'aucun message d'erreur n'est présent.

Balises

Nom Obligatoire Description
Contient Oui (2) Texte qui doit être trouvé dans le fichier pour que l'assertion ssoit vraie.
Fichier Oui Nom du fichier, avec son chemin d'accès, dans lequel les vérifications serons faites.
MessageEchec Non Message personalisé qui sera utilisé pour signler que l'assertion et fausse.
NeContientPas Oui (2) Texte qui ne doit pas être trouvé dans le fichier pour que l'assertion ssoit vraie.

Particularités

  1. Si une erreur est rencontrée, comme par exemple si un fichier inexistant est spécifié, l'assertion est considérée comme fausse.
  2. Il doit y avoir au moins une balise <Conttient> ou une balise <NeConttientPas>.
  3. Il peut t avoir plusieurs balises <Conttient> et <NeConttientPas> L'assertion est vraie si tous les textes spécifiées par les balises <Conttient> sont dans le fichier et si tous les textes spécifiés par les balises <NeConttientPas> ne sont pas trouvés dans le fichier .
