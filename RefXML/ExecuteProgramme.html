ExecuteProgramme

Description

ExecuteProgramme permet de lancer un programme dîsponible pour s'exécuter sur l'ordinateur où s'exécute JCAssertion.exe. JCAssertion va attendre tant que ce programme ne se terminera pas et vérifiera le code de retour que ce programme envoie au système d'exploitation. 0 est le code de retour qui est considéré comme un succès. Tous les autres codes sont considérés comme un échec de l'assertion.

Balises

Nom Obligatoire Description
Arguments Non Arguments à passer au programme exécuter selon la méthode de "ligne de commande".
MessageEchec Non Message personalisé qui sera utilisé pour signler que l'assertion et fausse.
Programme Obl Nom du programme à exécuter. Le nom doit contenir le chemin ou alors le programme doit être accessible à partir de tous les répertoires. (selon la variable d'environnement PATH)

Exemple

 <Assertion>
  <Description>Lancer un programme qui attend 30 secondes. </Description>
  <Type>ExecuteProgramme</Type>
  <Programme>{{VariableContenantLeChemin}}Attendre.exe</Programme>
  <Arguments>30</Arguments>
  <MessageEchec>Le programme attendre a mal fonctionné.</MessageEchec>
 </Assertion>
