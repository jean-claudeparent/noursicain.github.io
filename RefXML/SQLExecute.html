SQLExecute

Description

SQLExecute permet d'excéuter des commandes SQL qui modifient la base de données. On ne peutdonc pas faire de select avec cette commande. La mofication se fait sur la base de données définie avec la balise ConnectionOracle. Cette assertion est vraie sauf si la commande SQL provoque une erreur oracle.

Balises

Nom Obligatoire Description
SQL Obligatoire Commande SQL de modification de base de données à exécuter. Il peuut y avoir plus d'une balise SQL. Ne pas mettre de ";" à la fin de la commande.

Exemple

 <Assertion>
  <Description>Modifier une table. </Description>
  <Type>SQLExecute</Type>
  <SQL>delete from JCATest</SQL>
  <SQL>update JCATest set info = 'test'</SQL>
  </Assertion>
