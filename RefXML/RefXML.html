Référence du fichier XML d'assertion

Les types d'assertions suivants sont disponibles:

Type d'assertion Description
AssertSQL Définit une assertion composée d'une instruction SQL retournant une seule colonne dans une seule ligne, d'un opérateur et d'un  résultat attendu. Cela permet par exemple de vérifier qu'une rangée particulière existe dans une table oracle.
AssertXPath Définit une assertion composée d'une expression XPath retournant un nombre de noeud XML dans un fichier XML, d'un opérateur et d'un  résultat attendu. Cela permet par exemple de vérifier qu'un noeud XML particulier existe dans un fichier XML.
ChargeLOB Charge le contenu d'un fichier dans une colonne de type CLOB ou BLOB dans une table oracle pour une rangée qui existe déjà.
CompteFichiers Compte le nombre de fichiers dans un répertoire et fait une comparaison avec ce résultat. Par exemple vérifie qu'un répertoire contient plus de 400 fichiers
ConnectionOracle Définit les attributs d'une connection à une base de données oracle(via ODP) et permet d'ouvrir ou de fermer cette connections.
ContenuFichier Vérifie si un fichier contient et/ou ne contient pas un texte.
ExecuteProgramme Lance un programme et vérifie s'il s'est bien terminé.
ExporteLOB Écrit dans des fichiers le contenu d'une colonne de type CLOB ou BLOB pour une ou plusieurs rangées dans une table de la base de données.
FichierExiste Vérifie si un fichier existe.
MAJVariables Modifie la valeur d'une variable.
SQLExecute Exécute une commande SQL sur la connection définie par ConnectionOracle.
SubstituerVariablesFichier Lit un fichier modèle et écrit un fichier de sortie où les variables auront été remplacées par leur valeur.
